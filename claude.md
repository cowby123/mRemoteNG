# Claude AI 協作指南

## 溝通語言

**重要：本專案與 Claude AI 的所有溝通請使用繁體中文。**

## 專案資訊

- **專案名稱**: mRemoteNG
- **專案類型**: 遠端連線管理工具
- **開發語言**: C#

## 協作規範

### 1. 語言使用
- 所有對話、說明、註解請使用繁體中文
- 程式碼變數名稱、函數名稱保持英文
- 註解和文件說明使用中文

### 2. 溝通原則
- 清楚說明需求和問題
- 提供足夠的上下文資訊
- 確認變更前先詢問

### 3. 程式碼規範
- 遵循專案現有的程式碼風格
- 變更前先閱讀相關檔案
- 避免過度工程化
- 只進行必要的修改

### 4. Git 操作
- 不要自動提交程式碼，除非明確要求
- 提交訊息使用中文描述
- 遵循專案的分支策略
- **推送分支**: `apiserver`（預設推送到此分支）

## 常見任務

- 錯誤修復
- 功能開發
- 程式碼重構
- 程式碼審查
- 文件撰寫

## 注意事項

- 避免引入安全漏洞（SQL Injection、XSS、命令注入等）
- 不要刪除未使用的程式碼，除非確認無影響
- 保持向後相容性
- 測試變更是否影響現有功能
